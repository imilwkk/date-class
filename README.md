# Date Class (Java)

## Description
This project implements a custom Date class in Java.
It demonstrates how to create and work with date-related functionality
without using built-in date libraries.

---

## Features
- Create a date using day, month, and year
- Validate date values
- Compare two dates
- Display dates in a readable format

---

## Technologies Used
- Java

---

## How It Works
1. The Date class stores day, month, and year as fields.
2. Input values are validated to ensure the date is correct.
3. Methods allow comparison between dates.
4. Dates are printed in a clear and structured format.

---

## Example Usage
The program can:
- Create a date object
- Check if a date is valid
- Compare two different dates

---

## Purpose
This project was created as a university assignment to practice:
- Class design
- Basic date handling logic in Java
